 المخزن الدائري هو مصفوفة ذات حجم ثابت نتعامل معها و كأن نهايتها موصول ببدايتها
 خطوات الحل:
 1.إنشاء مخزن دائري يحتوي على:
 مصفوفة لتخزين البيانات..مؤشر كتابة(head)..مؤشر قراءة(tail)..عداد للعناصر (count)
 2.تم تجهيز المخزن بحيث يكون فارغ في البداية
 3.تم ادخال الاسم ليتم تخزينه في المخزن
 4.اذا كان المخزن يتسع لعدد الاحرف المدخلة..فتكون النتيجة هي الاسم المخل مع اضافة CE-ESY اما اذا كان عدد لايتسع لعدد الاحرف المظخلة..فتكون النتيجة هي buffer overflow ويظهر ايضا الاسم المدخل فقط
 5. بعد القراءة يصبح المخزن فارغ
